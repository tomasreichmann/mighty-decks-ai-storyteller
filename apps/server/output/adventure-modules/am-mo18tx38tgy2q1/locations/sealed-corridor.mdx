## Introduction

There is a sealed corridor. It looks like someone tried really hard to make sure nothing can enter or exit the corridor.

## Description

    -   🔎 It looks like there are more rooms there, but unsealing would require hiring a specialized crew with some serious equipment.
    -   🔎 You hear faint scratching from the other side.
    -   🏺 There are still two **Instant sealants** (Insulating Resource) stacked against the wall. You can use these to immediately seal a hull breach.
    -   💡 It will become possible to open this in Graveyard Bazaar
