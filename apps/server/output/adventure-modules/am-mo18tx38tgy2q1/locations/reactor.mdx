# Reactor

![](/api/adventure-artifacts/reactor-99836c3e8add9c720018.png)

## Introduction

Normally this is where the ship's power comes from. You can route it from here to different ship's system or even put it in overdrive and distribute some extra power.

## Description

    -   If fully operational:
        -   It should supply up to 3PU, but could be upgraded at specialized docks.
        -   You can put it in manual overdrive and supply up to 3 extra PU.
    -   It is running on minimal power, supplying just 1PU. All power is routed to life-support.
    -   🔎 The fuel is almost gone and there does not seem to be any more.
    -   🔎 Looks like there used to be some upgrades providing better energy output, but those have been uninstalled since.
    -   🔎 Looking inside the reactor is mesmerizing. You would have sworn you saw a dark shape in the reflection of its window behind you. There was nobody there.
